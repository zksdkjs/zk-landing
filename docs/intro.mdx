# Introduction

Build zk apps using the latest top-notch zk technology in JavaScript!

## Getting Started

zksdk is designed in a modular way, allowing you to download only the packages you need. Therefore, to get started, you must decide which package you want to use and follow its documentation.

If you would like to install the entire package:

```bash
npm i zkmerkle

or 

bun add zkmerkle

```
